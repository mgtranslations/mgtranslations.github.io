# 10分钟大学™  
克拉克马斯县园艺大师协会与俄勒冈州立大学扩展园艺大师™项目合作支持  

## 优秀植物推荐：多年生植物  
**S**: 阴生植物，**D**: 耐旱植物，**W**: 有益野生动物（鸟类、蜜蜂、蝴蝶、蜂鸟）  

### 植物名称及特点  
#### 蓍草  
**Achillea ‘Moonshine’**  
- **W**  
- 尺寸：3' x 3' (10年成熟：3' x 4')  
- 光照：全日照  
- 特点：夏季至初秋吸引授粉者的迷人花朵。  

#### 黄金花叶菖蒲  
**Acorus gramineus ‘Ogon’**  
- 尺寸：1' x 3' (10年成熟：1' x 5')  
- 光照：全日照，或轻度至开放阴影  
- 特点：引人注目的黄绿色条纹草状叶片，适合作为混合花坛的点缀、容器植物或水边植物。  

#### 林地银莲花  
**Anemone nemorosa**  
- **S, W**  
- 尺寸：4'' x 3' (10年成熟：4'' x 3.5')  
- 光照：全日照或轻度至开放阴影  
- 特点：低矮的扩展型植物，春季开出鲜艳的单瓣花。  

#### 布雷辛汉白玉苣  
**Bergenia ‘Bressingham White’**  
- 尺寸：1.5' x 3' (10年成熟：1.5' x 4')  
- 光照：全日照至轻度或开放阴影  
- 特点：纯白色钟形花与光滑的椭圆形叶子是这种玉苣的标志。  

#### 心叶玉苣  
**Bergenia cordifolia**  
- **S**  
- 尺寸：1.5' x 3' (10年成熟：1.5' x 4')  
- 光照：全日照至轻度或开放阴影  
- 特点：春季粗壮的花茎顶端簇拥着粉色至玫红色钟形花。  

#### 冬日光彩玉苣  
**Berginia cordifolia ‘Vinterglod’**  
- **S**  
- 尺寸：1.5' x 3' (10年成熟：1.5' x 4')  
- 光照：全日照至轻度或开放阴影  
- 特点：大胆光滑的绿叶伴随春季洋红色花朵，冬季叶片色彩丰富。  

#### 荷包牡丹  
**Dicentra formosa ‘Bacchanal’**  
- **S, W**  
- 尺寸：1' x 2' (10年成熟：1' x 4')  
- 光照：轻度开放或斑驳阴影  
- 特点：灰绿色叶丛顶部簇拥着酒红色心形花朵，是浪漫的林地植物。  

#### 洪堡县秋海棠  
**Epilobium septentrionale ‘Select Mattole’**  
- **D**  
- 尺寸：1' x 3' (10年成熟：1' x 5')  
- 光照：全日照  
- 特点：霜银色叶子与亮橙红色管状花形成鲜明对比，盛开时间从盛夏至深秋，植物呈整洁的扩展型但成丛状。  

#### 黑海淫羊藿  
**Epimedium ‘Black Sea’**  
- **W**  
- 尺寸：1'3'' x 3' (10年成熟：1'3'' x 5')  
- 光照：轻度至斑驳阴影  
- 特点：这种铁质多年生植物易于种植且外观迷人。从冬末到春初，细红色茎顶部会繁盛开出小巧的黄色花朵。  

#### 雷霆淫羊藿  
**Epimedium pinnatum ssp. Colchicum ‘Thunderbolt’**  
- 尺寸：1'3'' x 3' (10年成熟：1'3'' x 5')  
- 光照：轻度至开放阴影  
- 特点：秋季叶片呈现茄紫色，叶脉保持绿色，类似闪电图案，是这种常绿多年生植物的特点。  

#### 黄花淫羊藿  
**Epimedium × perralchicum**  
- **D, W**  
- 尺寸：1'3'' x 5' (10年成熟：1'3'' x 8')  
- 光照：轻度至斑驳阴影  
- 特点：冬末和早春开出黄油黄色花朵，植物快速生长且非常耐用。  

#### 混合淫羊藿  
**Epimedium x perralchicum ‘Frohnleiten’**  
- **S, D, W**  
- 尺寸：1'3'' x 5' (10年成熟：1'3'' x 8')  
- 光照：轻度至斑驳阴影  
- 特点：冬末黄油黄色花朵盛开，心形叶片呈现鲜亮绿色，并带有红色叶脉。  

#### 红色淫羊藿  
**Epimedium x rubrum**  
- **S, D, W**  
- 尺寸：1'3'' x 3' (10年成熟：1'3'' x 5')  
- 光照：轻度至斑驳阴影  
- 特点：坚韧持久的地被植物，三月末小型星形玫瑰红花朵接踵而至，随后叶片爆发出红色光泽。  

#### 心形淫羊藿  
**Epimedium x rubrum ‘Sweetheart’**  
- **D, W**  
- 尺寸：1'3'' x 2' (10年成熟：1'3'' x 3')  
- 光照：轻度至开放阴影  
- 特点：耐荫地被植物，耐深阴影但开花量减少。  

#### 樱桃红淫羊藿  
**Epimedium x versicolor ‘Cherry Tart’**  
- **D, W**  
- 尺寸：1'3'' x 2' (10年成熟：1'3'' x 3')  
- 光照：轻度至开放阴影  
- 特点：深玫瑰色花蕾盛开为粉红玫瑰花，黄柠檬中心杯围绕深黄色雄蕊，伴随巧克力紫叶片。  

#### 硫磺淫羊藿  
**Epimedium x versicolor ‘Sulphureum’**  
- **D, W**  
- 尺寸：1'3'' x 3' (10年成熟：1'3'' x 5')  
- 光照：轻度至斑驳阴影  
- 特点：春季开出淡黄色和黄油黄色双色花朵，是耐旱地被植物的优选。  

#### 沃利淫羊藿  
**Epimedium x warleyense**  
- **S, D, W**  
- 尺寸：1'3'' x 2' (10年成熟：1'3'' x 3')  
- 光照：轻度至斑驳阴影  
- 特点：早春铜红色花朵与红色叶片相得益彰。  

#### 橙色女王淫羊藿  
**Epimedium x warleyense ‘Orangekonigin’**  
- **D, W**  
- 尺寸：1'3'' x 2' (10年成熟：1'3'' x 3')  
- 光照：轻度至斑驳阴影  
- 特点：虽然生长较慢，最终覆盖较大面积，是理想地被植物。

### 关于优秀植物推荐（GPP）  
优秀植物推荐（Great Plant Picks, GPP）是Elisabeth C. Miller植物园的主要教育项目。2001年首次发布推荐名单，旨在为太平洋西北海洋气候地区的园艺师提供综合性的优秀植物选择。迄今为止，已为居住在美国俄勒冈州尤金至加拿大不列颠哥伦比亚省温哥华之间的卡斯卡德山脉以西地区的园丁筛选了超过1,000种杰出植物。  

#### GPP植物选择标准  
- 能适应多种土壤  
- 需适量或更少的额外水分  
- 耐热  
- 能耐寒（USDA 7区）  
- 不容易感染病虫害  
- 在花园中能提供超过一个季节的观赏价值  

此外，许多GPP植物还提供生态效益（吸引授粉者、为鸟类提供食物、栖息地等）。
