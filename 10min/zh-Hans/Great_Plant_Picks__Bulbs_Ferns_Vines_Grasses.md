## 优秀植物推荐：球根植物、蕨类植物、藤本植物和草类

S: 阴湿花园  
D: 耐旱植物  
W: 有益野生动物：鸟类、蜜蜂、蝴蝶、蜂鸟  

### 球根植物

#### 裸花水仙  
**名称**: Amaryllis belladonna  
**尺寸**: 1’6” x 1’6”  
**光照**: 全日照  
**特点**: 夏季中后期茎干枯萎，秋季出现18英寸长的喇叭状亮丽花朵。

#### 风花  
**名称**: Anemone blanda  
**尺寸**: 4” x 1’  
**光照**: 全日照  
**特点**: 春季早期开花，花朵呈雏菊状，有玫瑰色、白色和浓紫色靛蓝三色。

#### 库西克蓝花葱  
**名称**: Camassia cusickii  
**特点**: S, W  
**尺寸**: 3’ x 3’  
**光照**: 全日照到轻微或疏松阴影  
**特点**: 太平洋西北原生植物。球根植物，春末至初夏开出美丽的冰蓝到天蓝色花朵。

#### 蓝花葱  
**名称**: Camassia quamash  
**特点**: S, W  
**尺寸**: 2’ x 2’  
**光照**: 全日照到轻微或疏松阴影  
**特点**: 太平洋西北原生植物。易种植的球根植物，五月开出明亮的蓝色到紫色花穗。

#### 秋水仙  
**名称**: Colchicum autumnale  
**特点**: D  
**尺寸**: 1’ x 6”  
**光照**: 全日照  
**特点**: 秋季初，大型花瓶状的紫红色花朵从地面冒出，为花园增添一抹亮丽色彩。

#### 巨型秋水仙  
**名称**: Colchicum ‘The Giant’  
**特点**: D  
**尺寸**: 1’ x 4”  
**光照**: 全日照  
**特点**: 巨型花瓶状的紫红花朵，内瓣底部呈白色。这些壮丽的花朵在秋季初从地面冒出，为花园增添亮丽色彩。

#### 大花番红花  
**名称**: Crocus ‘Flower Record’  
**尺寸**: 5” x 6”  
**光照**: 全日照  
**特点**: 容易种植且长寿的杂交品种，花朵呈高脚杯状，颜色为深紫色。

#### 大花番红花  
**名称**: Crocus ‘Goldilocks’  
**尺寸**: 5” x 5”  
**光照**: 全日照  
**特点**: 长寿番红花，金色花朵上带有一抹酒红色。

#### 大花番红花  
**名称**: Crocus Jeanne d’ Arc  
**尺寸**: 5” x 9”  
**光照**: 全日照  
**特点**: 高脚杯状，惊艳的白色花朵外侧带有紫色斑点。长寿的番红花品种。

#### 大花番红花  
**名称**: Crocus ‘King of the Striped’  
**尺寸**: 5” x 9”  
**光照**: 全日照  
**特点**: 高脚杯状花朵，白色花瓣上覆盖独特的紫色条纹。橙色花药增添趣味。

#### 大花番红花  
**名称**: Crocus ‘Pickwick’  
**尺寸**: 5” x 9”  
**光照**: 全日照  
**特点**: 另一种美丽的番红花品种，白色花瓣上有紫色条纹。橙色花药增添趣味。

#### 大花番红花  
**名称**: Crocus ‘Yellow Mammoth’  
**尺寸**: 5” x 9”  
**光照**: 全日照  
**特点**: 高脚杯状花朵，呈浓郁的金盏花黄色，花药颜色相近。

#### 波斯紫罗兰  
**名称**: Cyclamen coum  
**特点**: S  
**尺寸**: 4” x 6”  
**光照**: 从浅阴到深阴  
**特点**: 地中海原生植物，夏季休眠。银币大小的叶子秋季晚期出现，从深绿色到纯银色不等。花朵颜色有白色、洋红色和粉色。

#### 常春藤叶紫罗兰  
**名称**: Cyclamen hederifolium  
**特点**: S  
**尺寸**: 6” x 6”  
**光照**: 从浅阴、疏阴到深阴  
**特点**: 植株在9月和10月开出粉色花朵，偶尔出现白色花朵，然后叶子开始长出。

#### 夏雪花  
**名称**: Leucojum aestivum ‘Gravetye Giant’  
**尺寸**: 1’6” x 1”  
**光照**: 全日照到轻微或疏松阴影  
**特点**: 球根多年生植物，直立的带状深绿色光泽叶子。春季无叶茎干可高达36英寸，最多开出8朵钟形微巧克力香味白花。

#### 迷你水仙  
**名称**: Narcissus ‘February Gold’  
**尺寸**: 12” x 12”  
**光照**: 全日照到浅阴  
**特点**: 美丽的三英寸金黄色花朵点缀在12英寸坚韧的茎干上。成组种植打造自然景观效果。

#### 迷你水仙  
**名称**: Narcissus ‘Jack Snipe’  
**尺寸**: 8” x 4”  
**光照**: 全日照到浅阴  
**特点**: 可爱的小型白色花朵，中心呈金黄色，上方有坚韧的8英寸茎干。

#### 白色喇叭水仙  
**名称**: Narcissus ‘Mount Hood’  
**尺寸**: 1’3” x 0.5’  
**光照**: 全日照到浅阴  
**特点**: 四英寸白色喇叭形花朵点缀在坚韧的18至20英寸茎干上。最好成组种植5到7个。

#### 金色喇叭水仙  
**名称**: Narcissus ‘Saint Keverne’  
**尺寸**: 1’3” x 0.5’  
**光照**: 全日照到浅阴  
**特点**: 大型金黄色喇叭形花朵点缀在坚韧的18至20英寸茎干上。

#### 迷你水仙  
**名称**: Narcissus ‘Tete-a-Tete’  
**尺寸**: 8” x 1’  
**光照**: 全日照到浅阴  
**特点**: 令人喜爱的水仙品种，花朵呈2.5英寸金黄色。

#### 白色兰花水仙  
**名称**: Narcissus ‘Thalia’  
**尺寸**: 1’ x 4”  
**光照**: 全日照到浅阴  
**特点**: 多头白花点缀在坚韧的茎干上。成组种植时效果迷人。

#### 紫色愚人葱  
**名称**: Triteleia ‘Corrina’  
**特点**: D, W  
**尺寸**: 1’ x 6”  
**光照**: 全日照  
**特点**: 西海岸原生植物。深紫色星形花簇在一英尺长茎干上绽放，春末至初夏由球根状球茎形成。

#### 黄色愚人葱  
**名称**: Triteleia ixioides ‘Starlight’  
**特点**: D  
**尺寸**: 1’ x 6”  
**光照**: 全日照  
**特点**: 西海岸原生植物。春末至初夏开出两英寸长淡黄色星形花朵。

#### 蓝色愚人葱  
**名称**: Triteleia laxa ‘Queen Fabiola’  
**特点**: D, W  
**尺寸**: 1’ x 6”  
**光照**: 全日照  
**特点**: 西海岸原生植物。紫色星形花簇在一英尺长茎干上绽放，春末至初夏由球茎形成。

### 蕨类植物

#### 铁线蕨  
**名称**: Asplenium trichomanes  
**特点**: S  
**尺寸**: 6” x 6”  
**光照**: 从浅阴到深阴  
**特点**: 太平洋西北原生植物。这种看似脆弱的蕨类植物有迷你的常绿叶子，形成可爱的绿色小簇，类似剑蕨的迷你版本。

#### 秋蕨  
**名称**: Dryopteris erythrosora  
**尺寸**: 2’ x 2’  
**光照**: 从浅阴到深阴  
**特点**: 秋蕨是一种易种植的常绿蕨类植物，新长出的叶子呈铜红色，与成熟的深绿色光泽叶子形成鲜明对比。

#### 光辉秋蕨  
**名称**: Dryopteris erythrosora ‘Brilliance’  
**尺寸**: 2’ x 2’  
**光照**: 全日照到轻微或疏松阴影  
**特点**: 'Brilliance'是秋蕨的一个品种。可在全日照条件下种植并定期浇水。阳光下颜色更为鲜艳。

#### 剑蕨  
**名称**: Polystichum munitum  
**特点**: S  
**尺寸**: 3’ x 3’  
**光照**: 从浅阴到深阴  
**特点**: 太平洋西北原生植物。剑蕨常绿，形成优美的弯曲植物状。耐干阴环境。

#### 软盾蕨  
**名称**: Polystichum setiferum Divisilobum Group  
**特点**: S  
**尺寸**: 1.5’ x 2’  
**光照**: 从浅阴到深阴  
**特点**: 细致分裂的常绿叶子优雅弯曲，形成隆起状植物。叶子弯曲时看起来像围绕植物中心旋转，呈现旋风效果。

### 草类植物

#### 羽毛芦苇草  
**名称**: Calamagrostis x acutiflora ‘Karl Foerster’  
**尺寸**: 5’ x 2’  
**光照**: 全日照  
**特点**: 两英尺高的叶子增加了结构感而不过于压迫。六月时花梗可长至五至六英尺，带有窄羽状粉红棕色花序，之后变成金色。

#### 蓝燕麦草  
**名称**: Helictotrichon sempervirens  
**尺寸**: 2’ x 2’  
**光照**: 全日照  
**特点**: 狭窄的尖状灰蓝色叶子形成小型建筑状常绿植物，夏季顶部有紫棕色种子头更显美观。

#### 巨型羽毛草  
**名称**: Stipa gigantea  
**尺寸**: 2’ x 4’  
**光照**: 全日照  
**特点**: 具有雕塑感的草类植物，18英寸叶子簇状，初夏种子头爆发性增长至六至八英尺，形成大型水滴状种子，干燥后呈蜂蜜金色。

### 藤本植物

#### 绯红荣耀藤  
**名称**: Vitis coignetiae  
**尺寸**: 30’ x 20’  
**光照**: 全日照到轻微或疏松阴影  
**特点**: 此藤本植物的名字说明了一切，秋季披上绯红色大叶子时确实十分壮观。心形叶子非常大，直径常达一英尺。

#### 紫叶葡萄藤  
**名称**: Vitis vinifera ‘Purpurea’  
**尺寸**: 20’ x 6’  
**光照**: 全日照  
**特点**: 一种普通酿酒葡萄的优选品种。叶子春季呈绿色带有酒红色光泽，秋季叶子变成饱和的深红色。

“优秀植物推荐”（Great Plant Picks，简称GPP）是伊丽莎白C.米勒植物园的主要教育项目。2001年首次推出推荐列表，旨在为太平洋西北海洋性气候地区打造全面的优秀植物组合。从俄勒冈州尤金市到加拿大不列颠哥伦比亚省温哥华，迄今已有超过1000种优秀植物被选中。

GPP植物入选标准包括：适应多种土壤条件，需中等或更少补充水分，耐热性强，在美国农业部第7区可耐寒，不易出现病虫害问题，并在花园中至少一年有多个季节表现出色。许多GPP植物还具有生态效益（为授粉昆虫、鸟类提供食物与栖息地等）。
