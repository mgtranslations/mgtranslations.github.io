## 优秀植物推荐：树木  
S：适合阴凉花园，D：耐旱植物，生态益处：鸟类、蜜蜂、蝴蝶、蜂鸟  

### 名称 S/D/W 10年成熟度 阳光/阴凉 突出特点  

#### 蜀枫  
**Acer circinatum**  
S 15’ x 15’ 20’ x 20’ 全日照到深阴凉  
太平洋西北地区原生植物。多茎树，秋季叶色美丽；在阳光下生长较直立，在阴凉处生长较开放。  

#### 红皮枫  
**Acer griseum**  
S 18’ x 15’ 30’ x 25’ 全日照，光到深阴凉  
适合小型花园的优良树种，全年树皮剥落美观，秋季叶色迷人。  

#### 三花枫  
**Acer trifolium**  
S 18’ x 15’ 30’ x 25’ 全日照，光到深阴凉  
吸引人的耐用树种，树皮和秋季叶色有趣。耐病虫害，根系不侵略性强。  

#### 彩绘枫  
**Acer ‘Warrenred’ PACIFIC SUNSET**  
20’ x 18’ 30’ x 25’ 全日照，光到深阴凉  
优秀的中型树种，适应城市艰难环境，秋季叶色美丽。  

#### 海棠叶阿萨树  
**Azara microphylla**  
15’ x 6’ 25’ x 12’ 全日照，光到斑驳阴凉  
令人惊叹的小型常绿树种。晚冬开花，花香如香草。适合种植在修剪成高枝的针叶树下。  

#### 白皮河桦  
**Betula nigra ‘Cully’ Heritage**  
S 30’ x 20’ 60’ x 30’ 全日照，光阴凉  
四季树皮美观，特别是冬季。抗桦树钻蛀虫。  

#### 柱状榉树  
**Carpinus betulus ‘Frans Fontaine’**  
25’ x 6’ 50’ x 20’ 全日照到光到深阴凉  
欧洲榉树选中的最佳街树品种。成熟后保持修长的柱状形态。即使在艰难环境中也能长寿。  

#### 金色梓树  
**Catalpa bignonioides ‘Aurea’**  
15’ x 20’ 20’ x 30’ 全日照，光到开放阴凉  
春末至初夏期间，巨大心形金色叶子从枝条上爆发，形成最具戏剧性的叶子展示之一。  

#### 山茱萸  
**Cornus mas**  
S, W 15’ x 15’ 20’ x 20’ 全日照，光到开放阴凉  
小型树种，冬季黄色花朵美丽，夏末可食用果实，秋季紫红色叶子。  

#### 紫叶扭曲榛树  
**Corylus avellana ‘Red Dragon’**  
D 6’ x 5’ 8’ x 7’ 全日照，光阴凉  
冬末装饰性花序，夏季暗紫红叶子，秋冬扭曲树形。抗榛树枯萎病。  

#### 土耳其榛树  
**Corylus colurna**  
25’ x 20’ 40’ x 30’ 全日照  
适合城市环境和干燥环境的优秀树种。其金字塔形态、彩色叶子和冬季花序装饰价值极高。抗榛树枯萎病。  

#### 拉瓦莱山楂  
**Crataegus x lavallei**  
W 15’ x 15’ 25’ x 25’ 全日照，光到开放阴凉  
耐城市环境的坚韧树种，绿叶持久到冬季。春季白花变成橙红色果实，冬季仍悬挂。  

#### 银杏树  
**Ginkgo biloba ’Autumn Gold’**  
15’ x 6’ 60’ x 30’ 全日照  
银杏进化已有超过2亿年历史。银杏叶形与女贞叶相似，边缘带有扇形纹理，因此得名。  

#### 金塔银杏  
**Ginkgo biloba ‘Blagon’ GOLD SPIRE**  
20’ x 6’ 30’ x 8’ 全日照，光到开放阴凉  
适合狭窄或小型地点的美丽树种。从其修长的形态到独特的扇形叶子，此品种非常优异。  

#### 普林斯顿哨兵银杏  
**Ginkgo biloba ‘PNI2720’ PRINCETON SENTRY**  
20’ x 12’ 60’ x 25’ 全日照，光到开放阴凉  
边缘扇形的明亮绿叶非常有特色。秋季叶子呈亮黄色，非常耀眼。  

#### 萨拉托加银杏  
**Ginkgo biloba ‘Saratoga’**  
12’ x 5’ 45’ x 25’ 全日照，光到开放阴凉  
雄性品种树，不会产生果实。自然形态良好，几乎无需修剪。秋季叶色美丽。  

#### 总统金银杏  
**Ginkgo biloba ‘The President’ PRESIDENTIAL GOLD**  
D 20’ x 15’ 50’ x 40’ 全日照，光到开放阴凉  
形态宽圆，分枝密集。出色的遮荫树种。叶子为明亮绿，边缘扇形，秋季变为金色。适应城市环境。  

#### 金雨树  
**Koelreuteria paniculata**  
18’ x 15’ 30’ x 25’ 全日照  
优秀的中型树种，适应城市艰难环境。夏中至夏末开出亮黄色的花簇，随后形成粉红红色种荚。无病虫害问题。  

#### 珊瑚阳光金雨树  
**Koelreuteria paniculata ‘Coral Sun’**  
15’ x 12’ 25’ x 20’ 全日照到光阴凉  
耐城市环境。此品种因其春季珊瑚粉红色叶子而被选中，叶子成熟后变为浅绿色，与深色珊瑚茎形成对比。  

#### 花蟠果海棠  
**Malus ‘Adirondack’**  
W 12’ x 8’ 18’ x 12’ 全日照  
深红色花蕾开放为晶莹白色带红色细纹的花朵，密布垂直形态的树枝。是所有海棠品种中花量最多的一种。  

#### 矮生山楂海棠  
**Malus toringo ssp. Sargentii ‘Tina’**  
W 4’ x 5’ 5’ x 6’ 全日照  
极矮的海棠品种，红色花蕾开为单一白花，随后形成鲜红色0.25英寸果实。适合小型花园或大型容器。  

#### 裂叶海棠  
**Malus transitoria ‘Schmidtcutleaf’ GOLDEN RAINDROPS**  
12’ x 14’ 20’ x 25’ 全日照  
美观且抗病的树种，白花覆盖密集，果实为金黄色，硬霜后略带红色。  

#### 波斯铁木  
**Parrotia persica**  
D 18’ x 15’ 40’ x 30’ 全日照，光到斑驳阴凉  
四季皆美。叶子、枝条结构和树皮都非常出色。叶子初春呈绿色带酒红色边缘，夏季变为中绿色，秋季逐渐变为青铜色、深红色、橙色，最后成为亮金色。  

#### 波斯铁木‘瓦妮莎’  
**Parrotia persica ‘Vanessa’**  
D 18’ x 8’ 30’ x 14’ 全日照，光到斑驳阴凉  
耐久的树种，四季皆美，叶子、枝条结构和树皮都非常出色。冬末红花，叶子萌芽前开花。秋季叶色从八月底开始变化，持续到十月。  

#### 红橡树  
**Quercus rubra**  
W 25’ x 20’ 50’ x 40’ 全日照，光到开放阴凉  
庄严的树种，是花园中最宏伟的橡树之一。枝条强壮，夏季深绿色叶子，秋季变为红色。  

#### 金槐树  
**Robinia pseudoacacia ‘Frisia’**  
25’ x 15’ 50’ x 30’ 全日照，光到开放阴凉  
极其耐用的树种。明亮的柠檬黄色叶子在阳光下最亮丽，阴凉处变为黄绿色。春末至夏初开白色芳香花。  

#### 轮树  
**Trochodendron aralioides**  
8’ x 6’ 25’ x 15’ 全日照到光阴凉  
单干树种，支撑水平分枝的阔叶常绿树种。  

#### 日本榉树  
**Zelkova serrata**  
D 30’ x 30’ 50’ x 50’ 全日照  
城市景观的优秀树种，独特的宽瓶形树形非常吸引人，与大多数其他树种不同。  

#### 日本榉树‘绿色瓶形’  
**Zelkova serrata ‘Green Vase’**  
30’ x 20’ 50’ x 30’ 全日照  
适合城市景观的优秀树种。与原种相似的瓶形，但生长速度几乎为原种的两倍。‘绿色瓶形’枝条结构优美，冬季轮廓迷人。  

#### 日本榉树‘村绿’  
**Zelkova serrata ‘Village Green’**  
25’ x 25’ 40’ x 40’ 全日照  
适合城市景观的优秀树种。耐风、污染、干旱和压实土壤。适合形成树木大道的理想选择。

### 优秀植物推荐（GPP）简介  
优秀植物推荐（GPP）是伊丽莎白·C·米勒植物园的主要教育项目。2001年首次推出列表，推荐为太平洋西北海洋气候地区打造全面植物组合。迄今为止，已为从美国俄勒冈州尤金市到加拿大不列颠哥伦比亚省温哥华的园丁选出超过1,000种优秀植物。  

GPP植物满足以下标准：适应多种土壤环境，需中等或更少的额外水分，耐热，耐寒（符合美国农业部7号种植区标准），不易受疾病或害虫侵扰，并在花园中提供多个季节的欣赏价值。许多GPP植物还提供生态益处（为传粉者、鸟类、野生动物提供食物或栖息地等）。
