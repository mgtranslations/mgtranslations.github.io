# Лучшие цветы для пчел — проверено садоводами [cite: 1]

Этот список включает растения, доказанные доктором Гордоном Фрэнки из Университета Калифорнии и его коллегами как лучшие для привлечения пчел [cite: 1]. Растения, привлекающие местных пчел в Орегоне, основаны на исследованиях Аарона Андерсона из Лаборатории экологии садов Университета штата Орегон [cite: 2]. Источники: California Bees & Blooms Франки и др., blogs.oregonstate.edu/gardenecologylab/[cite: 3].
