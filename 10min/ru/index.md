# Раздаточные материалы 10-минутного университета™ – Ассоциация мастеров-садоводов округа Клакамас™

**10-минутный университет™** — это зарегистрированная торговая марка коротких занятий по важным вопросам садоводства, которые проводят мастера-садоводы Службы расширения Орегонского государственного университета. *Все раздаточные материалы были проверены преподавателями Службы расширения ОСУ на достоверность информации.*
