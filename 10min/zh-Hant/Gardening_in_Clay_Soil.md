## 在黏土土壤中種植  

### 介紹  
土壤由岩石分解和有機物質構成。健康的土壤在體積上應該包含以下成分：45%岩石礦物質、5%有機物質、50%空隙供水和空氣流通。土壤的重要功能包括：  
- 為細菌、真菌、線蟲、原生動物和蚯蚓等生物提供棲息地；  
- 為植物根部提供支撐；  
- 作為植物生長所需養分的儲存庫；  
- 水的儲存和淨化。  

土壤中的礦物質顆粒包括沙粒、粉砂和黏土。沙粒肉眼可見且摸起來粗糙；粉砂顆粒較小（類似於麵粉顆粒），摸起來光滑，形狀像微型岩石。黏土顆粒肉眼不可見，通常呈扁平狀。大多數土壤是沙粒、粉砂和黏土的混合物。壤土則是這三種顆粒比例大致相等的混合物。  

土壤的質地反映了沙粒、粉砂和黏土的比例；良好的土壤結構依賴於活生物體（如細菌和真菌）以及腐殖質（分解的有機物質），它們將顆粒黏合成土塊（peds），這些土塊是土壤的基本構成單位。

### 黏土土壤與其他土壤類型的比較  

| 土壤類型        | 壤土 | 沙壤土 | 粉砂壤土 | 黏土和黏壤土 |  
| ----------------- | ---- | ------ | -------- | ------------ |  
| **含水能力**     | 中等 | 低-中等 | 高      | 高           |  
| **肥力**         | 中等 | 低      | 中等    | 高           |  
| **透水性**       | 中等 | 高      | 低-中等 | 低           |  
| **手感**         | --   | 粗糙   | 濕潤時柔軟不粘 | 乾時硬，濕時粘，能被捏成條狀 |

### 土壤科學的啟示  

#### 肥力  
- 黏土土壤富含植物養分。黏土顆粒帶有負電荷，能吸附帶正電荷的養分，如鈣、鉀、鎂、鐵等。  

#### 透水性  
- 濕潤的土壤以及細質地土壤（含有大量黏土或粉砂）容易壓實。壓實的土壤中土塊變得平坦，阻礙水和根的流動。壓實的黏土土壤很難修復。  
- 健康的土壤中土塊之間有大量空隙（宏孔），能促進水、氣體和根的流動。更多宏孔能改善排水性能。  

#### 含水能力  
- 小孔或微孔（土塊內的空隙）賦予土壤儲水能力。  

#### 攪動土壤  
- 挖掘和耕作會破壞土塊，但也會將空氣引入宏孔。過度耕作會降低真菌菌絲的功能，殺死有益線蟲，通常導致土壤壓實。  

#### 土壤生命  
- 土壤中充滿生命——許多是對植物有益的生物。含有較高有機物質的土壤通常含有更多微生物。土壤微生物在70°F至100°F之間最活躍，而蚯蚓在大約50°F時最活躍。  
- 植物根部會分泌一種稱為根滲物的物質，其中包含糖、維生素、氧氣和其他化合物。根滲物吸引微生物到植物根部周圍。研究表明，根滲物的組成因植物種類而異，並影響植物周圍真菌群落的構成。  

#### 有益微生物  
- 根瘤菌能從空氣中捕獲氮供植物使用。  
- 有益線蟲對干擾非常敏感，耕作通常會殺死它們，進而擾亂整個土壤食物網。  
- 許多種類的真菌與植物根部形成有益的聯繫（稱為菌根）。真菌菌絲就像長長的根毛，幫助植物吸收水分和土壤養分。菌根還通過聚集土壤顆粒形成土塊來改善土壤結構。

### 如何管理黏土土壤  

#### 測試土壤  
- 在添加改良劑之前，先測試土壤。只有在土壤測試顯示缺乏養分或植物生長不良時，才需要施肥。記住，黏土土壤本身富含養分。在威拉米特山谷，大多數樹木、灌木和多年生植物不需要額外施肥。  

#### 保持活根生長  
- 活植物通過其根部滲物收集陽光的能量並支持土壤微生物群落。研究表明，根滲物影響土壤微生物的構成。種植多種植物有助於最大化土壤生命的多樣性，從而建立健康的土壤。  

#### 減少干擾  
- 任何干擾土壤的行為（如走在上面、挖掘、耕作）都可能破壞或壓實土塊，減少對透水性和含水能力至關重要的空隙。  

#### 保持覆蓋  
- 使用覆蓋物來調節土壤溫度，減少蒸發，並抑制雜草生長。有機覆蓋物分解後會增加土壤中的有機物質——較粗的材料能促進空氣流通，而密度較大的物質（如紙板）會阻礙氣體交換。樹木修剪的木屑（4-6英寸）是一種良好的覆蓋物，它既能促進空氣流通，又能慢慢分解為土壤增加有機物質。  

#### 餵養土壤微生物  
- 有機物質（OM）是土壤食物網的基礎；添加有機物質能滋養土壤微生物。選擇經過適當堆肥的有機物質，避免含有高鹽或重金屬等不良成分。可以將堆肥的葉子和庭院廢料混合進土壤中，或者鋪設一層1/2到1英寸的薄層，為微生物活動創造有利環境。  

#### 為本季準備種植床  
- 如果需要為一年生蔬菜作物準備新的種植床，可以將堆肥的有機物質混合到整個種植床中。這應能立即改善土壤的可操作性和肥力。然而，這種益處是短期的，因為有機物質會迅速分解，未使用的養分可能會流失，土壤也會沉降。  

#### 從過去的改變  
- “只需添加有機物質！”的口號已被修改。過去，我們建議最初添加2-4英寸的有機物質，隨後減少添加量。但這已改變。最近對多年添加有機物質的城市花園的研究表明，磷酸鹽和有機物質的水平已達到不健康的程度。高磷酸鹽水平會抑制植物對鐵的吸收以及有益菌根的形成，並導致河流污染。如果從商業來源購買有機物質，俄勒岡州農業部會分析其養分含量。如果您長期向土壤添加有機物質，請確保購買低磷含量的有機物質。
